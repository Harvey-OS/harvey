Ghostscript bug report form

Before using this form, please verify that your problem can be reproduced using Ghostscript alone, without using any front end viewer such as GSview, Ghostview, or gv. If your problem only shows up when you use a viewer, please report the problem to the author of the viewer.

If the problem is really in Ghostscript, please check the "known bugs" notice at

http://www.cs.wisc.edu/~ghost/aladdin/relnotes/gs###/index.html

where "###" is the unpunctuated version number of Ghostscript's latest release; and then "Reporting bugs" in the documentation for new users. For other information, see the Ghostscript overview.

If the problem is not already known, please fill out this form and e-mail it to bug-gs@aladdin.com. You must fill out all applicable items in this form. In particular, you must include all data (PostScript files, PDF files, batch/script files, and/or fonts) involved in the problem, either by giving a URL from which we can download the data, or by including the data with the report itself. If you don't include this information, we can't investigate the problem.

Fully completed problem reports help us improve Ghostscript. Thanks in advance.
