Signup form for notification of Ghostscript test releases

This document contains a form that you can fill out if you would like to be informed by e-mail when new pre-public-release Ghostscript filesets are made available.

If you are interested in testing pre-public-release Ghostscript filesets, please read this document. Then if you would like to be notified when new filesets are posted, please fill out this questionnaire and e-mail as indicated below.

We don't share any of the above information with anyone else, with two exceptions:

  • If someone needs help getting the Ghostscript files (because they have a slow modem or no FTP access), we will sometimes refer them to a nearby user from this list. In this case, we give them only the name, city, and e-mail address.
  • One of the documentation files included with Ghostscript includes a thank-you for all the people who have helped develop and test it.

If you prefer that we not use your name even in these ways, please let us know.

If you are interested in testing Ghostscript on the Macintosh platform, please mail the completed questionnaire to mac-gs-test-request@aladdin.com. If you are interested in testing Ghostscript on platforms other than the Macintosh, please mail the completed questionnaire to gs-test-request@aladdin.com. (Of course, you can mail it to both places if appropriate.)

Please do not distribute pre-public-release filesets outside your organization. You are welcome to direct people to http://www.cs.wisc.edu/~ghost/aladdin/tester.html, where they can find out about getting these filesets.

Thanks again for being willing to help improve Ghostscript.
